# ⚡ Household Energy Consumption Prediction

## 📌 Project Overview

This project predicts household energy consumption using Machine Learning Regression techniques. A synthetic dataset was created to simulate household energy usage under different environmental conditions and appliance usage scenarios.

The project demonstrates the complete machine learning workflow, including data generation, exploratory data analysis (EDA), data preprocessing, feature engineering, model training, and model evaluation.

---

## 🎯 Problem Statement

Energy consumption varies depending on several factors such as temperature, humidity, appliance usage, and weekends. The objective of this project is to predict household energy consumption based on these factors.

---

## 📂 Dataset

This project uses a **synthetic dataset** generated using Python.

### Features

| Feature      | Description                                              |
| ------------ | -------------------------------------------------------- |
| temperature  | Ambient temperature (°C)                                 |
| humidity     | Relative humidity (%)                                    |
| fans_on      | Number of fans currently running                         |
| ac_on        | Air conditioner status (0 = Off, 1 = On)                 |
| lights_on    | Number of lights switched on                             |
| weekend      | Weekend indicator (0 = No, 1 = Yes)                      |
| energy_units | Estimated household energy consumption (Target Variable) |

* Number of Records: **1000**
* Dataset Type: Synthetic
* Problem Type: Regression
