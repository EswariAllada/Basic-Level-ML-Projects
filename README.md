#  Beginner Machine Learning Projects

Welcome to my **Basic level Machine Learning Projects** repository!

This repository contains four beginner-friendly end-to-end machine learning projects built using **Python** and **Scikit-learn**. Each project follows a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and prediction.

##  Projects

###  Energy Consumption Prediction

Predicts household energy consumption using environmental and appliance-related features.

###  Insurance Charges Prediction

Predicts medical insurance charges based on demographic and lifestyle information.

###  Wine Quality Prediction

Classifies wine quality using physicochemical properties with a machine learning classification model.

###  Store Sales Prediction

Forecasts store sales using historical sales data and feature engineering techniques.
